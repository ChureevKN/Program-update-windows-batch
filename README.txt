Изначально было сделано несколько разных программ, но
в работу пошли два "пакета": четвертый и пятый.

===========================================

Пакет 4, файл 4.bat - Переход на HostCrypt.
Ставит ярлык HostCrypt в автозапуск и на рабочий стол, удаляет старую программу
CypherBase.

Пользователей заранее предупреждали об обновлении - закроется
CypherBase и включится новая программа.

Если нет версии 1.3 ещё - запускает Пакет 5 и ждёт, потом уже ставит автозапуск.
Запускается на рабочих местах при открытии другой программы, которую открывают
все пользователи каждый день (распространяются пакеты через сервер антивируса,
запускаются пользователем, незаметно для него, по ссылке с уже установленной на
всех компьютерах программе), после чего на компьютере проводится проверка
условий и если этот компьютер попадает под определение того, которому нужен этот
функционал, на нём либо сразу удаляется старая программа CypherBase и настраивается
автозапуск уже установленной заранее новой программы HostCrypt, либо сначала
вызывается "Пакет 5" для правильной установки HostCrypt без его включения в работу,
чтобы он не запускался сам и пользователь не видел его на рабочем столе и в "Пуск",
а затем продолжается исполнение этого "пакета" (удаляется старое, нового - автозапуск).

===========================================

Пакет 5, файл 5.bat - Невидимая установка новой программы HostCrypt.
Ставит с нуля или поверх старой версии HostCrypt 1.2, отключает автозапуск,
удаляет свой ярлык с рабочего стола и из меню "Пуск", оставляет работать
старую программу, CypherBase, так что пользователь не запустит новую программу
до нужного момента перехода со старой на новую (когда переход будет осуществлен
на сервере).

Проверка условий проводится перед запуском - не запускается, если уже есть файл-флаг
установки последней версии 1.3.
Если установлен CypherBase или HostCrypt 1.2, ставит сперва 1.2 (поверх, во втором
случае), ставит в хранилища необходимые сертификаты, затем ставит HostCrypt 1.3,
ставит файл с предварительно выставленными правильными настройками HostCrypt.

После установки ставит файл-флаг установки версии 1.3.

их версии, содержимое их файлов - изменены, с сохранением логики работы
*.BAT-файлов.
